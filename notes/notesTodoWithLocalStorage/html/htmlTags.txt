<body>

<h2>
<h3>

Anchor Tag: <a></a>
<a href="https://www.w3schools.com">This is a link</a>

Image tag: <img>
<img src="img.jpg" alt="img.com" width="104" height="142">

Attributes:

src      // source ref to a file <img src="img_girl.jpg">
width    // <img src="img_girl.jpg" width="500" height="600">
height
alt      // provides an alternate text to an image in case it fails loading
style    // add styles to an element like font, colur etc
eg: <p style = "color: red">    </p>
title

font-size         // <h1 style="font-size:300%;">This is a heading</h1> // text size
text-align        // <h1 style="text-align:center;">Centered Heading</h1> // alignment of text
